# 古罗马视觉参考

> 博尔赫斯《永生》场景参考

---

## 时代背景

- **时代**：公元3世纪（狄奥克莱西安诺皇帝，284-305年在位）
- **地点**：埃及、底比斯、红海沿岸
- **文化**：古罗马帝国晚期

## 场景元素

### 底比斯城

```
ancient egyptian city, thebes,
stone walls, monumental gates,
roman soldiers, marketplace,
desert background, nile river
```

### 沙漠旅程

```
vast egyptian desert,
endless sand dunes,
scorching sun,
lone traveler,
caravan in distance
```

### 永生者之城

```
ancient ruined city,
crumbling temples,
overgrown structures,
mysterious atmosphere,
dimly lit passages,
ancient statues
```
