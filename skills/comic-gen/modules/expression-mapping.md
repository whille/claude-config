# 表情关键词映射

> 将情绪描述转换为视觉表情 Prompt

---

## 基础情绪

| 情绪 | 中文关键词 | 英文 Prompt |
|------|-----------|-------------|
| 喜 | 高兴、欢喜、笑、愉悦 | `happy, slight smile, joyful expression` |
| 大喜 | 哈哈大笑、喜不自胜 | `laughing, wide smile, joyful, eyes crinkled` |
| 怒 | 生气、愤怒、冷笑、恼怒 | `angry, frowning, stern expression, furrowed brows` |
| 大怒 | 勃然大怒、暴怒 | `furious, enraged, intense anger, clenched jaw` |
| 哀 | 悲伤、哭泣、忧伤、落泪 | `sad, melancholic, teary eyes, downcast` |
| 大悲 | 痛哭、嚎啕 | `crying, tears streaming, sobbing` |
| 惊 | 惊讶、震惊、愣住、呆住 | `surprised, wide eyes, shocked expression` |
| 大惊 | 大惊失色、惊恐 | `terrified, mouth agape, frozen in shock` |
| 思 | 思索、沉吟、低头沉思 | `thoughtful, looking down, pondering, hand on chin` |
| 疑 | 疑惑、不解、皱眉 | `puzzled, questioning look, slight frown` |
| 怕 | 害怕、恐惧、畏惧 | `fearful, trembling, pale face` |
| 羞 | 羞涩、脸红、低头 | `shy, blushing, averting eyes` |

## 使用原则

1. **原文优先**：原文有具体描述时直接翻译
2. **避免过度**：不用夸张词汇，保持东方含蓄美
3. **配合姿态**：表情与动作协调
4. **人物性格**：考虑角色性格特点

### 错误示例

| ❌ 错误 | ✅ 正确 |
|---------|---------|
| `ecstatic, jumping for joy` | `happy, slight smile` |
| `raging, screaming` | `angry, frowning` |
| `bawling, wailing` | `teary-eyed, sad` |
