# 转场规则

> 分镜脚本的镜头切换序列与节奏控制

---

## 运镜控制（新增）

> 详见 `camera-movement.md`

每帧 Prompt 应包含运镜词，增强动态感：

| 运镜 | 英文 Prompt | 适用场景 |
|------|-------------|----------|
| 推镜头 | `Slow camera push-in` | 情绪聚焦 |
| 拉镜头 | `Camera pull-back` | 环境交代 |
| 跟随镜头 | `Tracking shot` | 行走、追逐 |
| 静态镜头 | `Static shot` | 对话、静谧 |
| 左/右摇 | `Pan left / Pan right` | 环境展示 |

**组合示例**：`medium shot, slow camera push-in`

---

## 标准序列

### 三帧序列（基础）

```
全景（交代环境）→ 中景（人物互动）→ 近景（表情细节）
```

### 五帧序列（完整）

```
全景（环境）→ 中景（动作）→ 近景（表情）→ 特写（细节）→ 远景（收尾）
```

---

## 景别关键词

| 景别 | 英文术语 | 使用场景 | Prompt 关键词 |
|------|----------|----------|--------------|
| 全景 | Wide shot | 交代环境 | `wide shot`, `full view`, `establishing shot` |
| 中景 | Medium shot | 人物互动 | `medium shot`, `waist-up view` |
| 近景 | Close-up shot | 表情细节 | `close-up`, `face shot`, `head and shoulders` |
| 特写 | Extreme close-up | 情绪高潮/细节 | `extreme close-up`, `detail shot` |
| 远景 | Long shot | 收尾/氛围 | `long shot`, `distant view` |
| 主观视角 | POV | 第一人称 | `POV shot`, `first-person view` |
| 俯视 | High angle | 上帝视角 | `high angle`, `bird's eye view` |
| 仰视 | Low angle | 崇高感 | `low angle`, `looking up` |

## 转场类型

### 硬切（Cut）

直接切换，最常用。

| 适用场景 | 说明 |
|----------|------|
| 对话 | A 说 → B 说 |
| 动作 | 动作开始 → 动作结束 |
| 时间流逝 | 白天 → 黑夜 |

### 软切（Dissolve）

渐变过渡，温和。

| 适用场景 | 说明 |
|----------|------|
| 回忆 | 现在 → 过去 |
| 梦境 | 现实 → 梦境 |
| 时间流逝 | 季节变化 |

### 空镜（Insert）

插入场景空镜头，调整节奏。

| 适用场景 | 说明 |
|----------|------|
| 情绪过渡 | 悲伤 → 空镜 → 坚定 |
| 时间流逝 | 花开、落叶、月升 |
| 氛围渲染 | 风景、天气 |

## 情绪节奏控制

### 紧张场景

```
近景（紧张表情）→ 特写（细节）→ 近景（反应）→ 中景（动作）
```

**特点**：短镜头，快节奏

### 悲伤场景

```
中景（人物）→ 远景（孤单）→ 空镜（环境）→ 近景（表情）
```

**特点**：长镜头，慢节奏

### 欢快场景

```
全景（热闹）→ 中景（互动）→ 中景（互动）→ 全景（氛围）
```

**特点**：中等镜头，中等节奏

---

## 对话序列

### 双人对话

```
全景（两人位置）→ 中景（A 说话）→ 中景（B 反应）→ 近景（A 表情）
```

### 多人对话

```
全景（群体）→ 中景（主要说话者）→ 中景（其他反应）→ 全景（群体）
```

---

## 动作序列

### 打斗

```
全景（对峙）→ 中景（出招）→ 近景（击中）→ 特写（表情）→ 远景（结果）
```

### 追逐

```
全景（起跑）→ 中景（奔跑）→ 主观视角（追逐感）→ 全景（结果）
```

### 书写/阅读

```
中景（人物）→ 特写（手部动作）→ 特写（文字内容）→ 近景（表情）
```

---

## 场景转换规则

### 同地点时间流逝

```
全景（环境）→ 空镜（时间符号：日出/日落）→ 全景（环境变化）
```

### 不同地点

```
空镜（原地点特征）→ 空镜（新地点特征）→ 全景（新场景）
```

### 回忆插入

```
近景（现在表情）→ 软切 → 全景（回忆场景）→ ... → 软切 → 近景（回归现在）
```

## 分镜脚本模板

```markdown
## 第 X 帧

- **景别**：全景/中景/近景/特写
- **内容**：画面描述
- **动作**：人物动作
- **表情**：人物表情
- **对白**：（如有）
- **转场**：硬切/软切/空镜
- **时长**：短/中/长

### Prompt 片段

```
{景别关键词},
{内容描述},
{人物动作},
{表情描述},
{环境氛围},
{时代风格}
```
```

## 特殊转场

### 蒙太奇

快速切换，暗示时间流逝或平行事件。

```
动作1（短）→ 动作2（短）→ 动作3（短）→ 结果
```

### 对比蒙太奇

对比两个场景，强调差异。

```
富贵场景 → 贫穷场景
快乐 → 悲伤
拥挤 → 孤单
```

### 平行剪辑

交替展示两条线索。

```
线索A（短）→ 线索B（短）→ 线索A（短）→ 线索B（短）→ 汇合
```
