# 智能分镜增强

> 提升分镜质量与叙事完整性

---

## 问题背景

**现象**：
1. 帧数过少导致情节跳跃
2. 关键情节点遗漏
3. 分镜节奏不合理

**根因**：
1. 单阶段分镜，缺乏细化
2. 未区分情节重要程度
3. 缺少密度控制机制

**解决方案**：建立两阶段分镜机制，智能控制分镜密度。

---

## 两阶段分镜流程

### 阶段一：粗分镜（节点级）

**目标**：将章节拆解为情节点（Node）

**输出**：情节点列表

```markdown
## 情节点列表

1. 【开场】林冲在草料场值班
2. 【转折】发现草料场被烧
3. 【高潮】山神庙杀敌
4. 【收尾】投奔梁山
```

**规则**：
- 每个情节点对应 1 个叙事单元
- 标记情节类型（开场/发展/转折/高潮/收尾）
- 估算时长占比

### 阶段二：细化分镜（帧级）

**目标**：将每个情节点拆解为具体帧

**输出**：分镜脚本

```markdown
## 情节点 1：林冲在草料场值班

| 帧号 | 景别 | 内容 | 时长 |
|------|------|------|------|
| 1 | 全景 | 草料场全景，风雪中 | 长 |
| 2 | 中景 | 林冲巡视草料 | 中 |
| 3 | 近景 | 林冲表情疲惫 | 短 |
```

## 分镜密度控制

### 情节类型 → 帧数映射

| 情节类型 | 帧数 | 说明 |
|----------|------|------|
| 开场 | 2-3 帧 | 环境交代 + 人物引入 |
| 过渡 | 1 帧 | 空镜或环境 |
| 对话 | 2-3 帧 | 说话 + 反应 + 回应 |
| 动作 | 3-5 帧 | 起势 + 过程 + 结果 |
| 高潮 | 5-7 帧 | 铺垫 + 爆发 + 余韵 |
| 收尾 | 2-3 帧 | 结果 + 氛围 |

### 场景复杂度 → 帧数加成

| 复杂度 | 加成 | 示例 |
|--------|------|------|
| 单人场景 | 基础帧数 | 林冲独处 |
| 双人场景 | +1 帧 | 对话、互动 |
| 多人场景 | +2 帧 | 群体活动、战斗 |
| 复杂动作 | +2 帧 | 打斗、追逐 |

### 情绪强度 → 帧数加成

| 强度 | 加成 | 示例 |
|------|------|------|
| 平静 | 基础帧数 | 日常对话 |
| 中等 | +1 帧 | 争论、思考 |
| 强烈 | +2 帧 | 悲伤、愤怒、恐惧 |

---

## 帧类型定义

| 类型 | 说明 | Prompt 特征 |
|------|------|-------------|
| 环境帧 | 交代场景 | 全景、无人物、氛围词 |
| 人物帧 | 展示人物 | 中景/近景、人物描述 |
| 动作帧 | 展示动作 | 中景、动作词、动态词 |
| 表情帧 | 展示情绪 | 近景/特写、表情词 |
| 细节帧 | 展示细节 | 特写、物品描述 |
| 转场帧 | 场景转换 | 空镜、过渡元素 |

---

## 分镜节奏控制

### 标准节奏

```
环境帧 → 人物帧 → 动作帧 → 表情帧 → （转场帧）
```

### 紧张节奏

```
动作帧 → 动作帧 → 动作帧 → 表情帧
```
（短镜头，快切换）

### 舒缓节奏

```
环境帧 → 人物帧 → 环境帧
```
（长镜头，慢切换）

## 算法实现

### 帧数估算公式

```python
def estimate_frame_count(chapter):
    """估算章节总帧数"""
    # 基础帧数
    base = len(chapter.scene_changes) * 2
    base += len(chapter.dialogues) * 2
    base += len(chapter.action_sequences) * 3

    # 情节加成
    for node in chapter.nodes:
        if node.type == "高潮":
            base += 3
        elif node.type == "转折":
            base += 1

    # 复杂度加成
    for scene in chapter.scenes:
        if scene.complexity == "多人":
            base += 2
        elif scene.complexity == "双人":
            base += 1

    return base
```

### 分镜生成流程

```python
def generate_storyboard(chapter):
    """生成完整分镜"""
    frames = []

    # 阶段一：粗分镜
    nodes = extract_plot_nodes(chapter)

    # 阶段二：细化分镜
    for node in nodes:
        node_frames = expand_node_to_frames(node)
        frames.extend(node_frames)

    # 验证
    assert len(frames) >= estimate_frame_count(chapter) * 0.8

    return frames
```

## 最佳实践

### 避免

- ❌ 单情节点只有 1 帧（信息不足）
- ❌ 连续 3 帧同景别（视觉单调）
- ❌ 高潮场景帧数不足（叙事不完整）

### 推荐

- ✅ 每情节点至少 2 帧
- ✅ 景别交替：全景→中景→近景
- ✅ 高潮场景多帧展示
