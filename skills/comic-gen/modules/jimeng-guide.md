## 使用流程

### 基础生成

```
1. 访问 jimeng.jianying.com
2. 登录（抖音/手机号/头条账号）
3. 点击「AI 绘画」
4. 输入 Prompt（支持中文）
5. 选择比例（推荐 16:9 漫画横版）
6. 点击生成
7. 等待 10-30 秒
8. 下载图片
```

### 角色一致性

```
1. 生成满意的首图
2. 点击图片下方「保持角色一致性」
3. 系统自动锁定角色特征
4. 修改 Prompt（改变动作/场景）
5. 生成后续帧
6. 角色形象保持一致
```

### 风格一致性

```
1. 生成满意的首图
2. 点击「保持风格一致」
3. 后续帧自动继承风格
```

## Prompt 模板

### 中文模板（即梦原生支持）

```
【完整 Prompt 模板】
{时代背景} {题材} {艺术风格}，
{景别}，
{场景描述}，
{人物描述}，
{动作描述}，
{表情描述}，
```

```
缓步前行，低头沉思，
忧伤氛围，柔和光线
```

## 常见问题

### Q: 角色一致性效果不稳定？

**解决方案**：
1. 首图选择清晰、无遮挡的图片
2. 角色特征词保持固定
3. 避免大角度变化（如正面→背面）

### Q: 风格差异明显？

**解决方案**：
1. 开启「风格一致性」功能
2. 固定风格前缀词
3. 使用相同的比例和参数

### Q: 生成速度慢？

**原因**：高峰期排队
**解决**：错峰使用（早/晚）

## 与 comic-gen 集成

在 comic-gen 工作流中：

| 步骤 | 使用即梦 |
|------|----------|
| Step 4: 图片生成 | 即梦网页生成 |
| Step 5: 构建输出 | 下载后整理到项目目录 |

**操作流程**：
1. comic-gen 生成分镜脚本和 Prompt
2. 复制 Prompt 到即梦
3. 生成首图，开启角色一致性
4. 批量生成后续帧
5. 下载并整理到项目目录
