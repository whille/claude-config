## 【新增】背景约束术语

### 空间类型与背景（通用）

| 空间类型 | 特征 | 背景关键词 |
|----------|------|-----------|
| **封闭空间** | 有围墙/墙壁边界 | `within enclosed walls`, `no distant view`, `boundary visible` |
| **院内场景** | 院子、庭院 | `enclosed courtyard`, `within compound walls` |
| **园林内部** | 中国古典园林 | `within garden walls`, `no distant mountains` |
| **室内场景** | 房屋内部 | `interior`, `no outdoor view`, `inside building` |
| **城市街巷** | 街道两侧有建筑 | `city street`, `buildings lining the street`, `no open landscape` |
| **开放空间** | 无边界遮挡 | `open landscape`, `distant mountains acceptable` |

### 背景边界术语

| 术语 | 说明 | 使用场景 |
|------|------|----------|
| `within walls` | 围墙内 | 封闭院落、园林 |
| `enclosed courtyard` | 封闭院落 | 院落场景 |
| `walled enclosure` | 围墙边界 | 强调边界 |
| `no distant view` | 无远景 | 封闭空间 |
| `boundary visible` | 边界可见 | 强调空间有限 |
| `interior setting` | 室内 | 建筑内部 |

### "山"类词汇翻译（关键）

| 原文用词类型 | 实指 | 视觉关键词 | ❌ 错误翻译 |
|----------|------|-----------|-------------|
| 假山、怪石 | 人工假山 | `artificial rockery`, `garden rocks` | ❌ `mountain` |
| 山石、叠石 | 园林石景 | `garden stones`, `stacked rocks` | ❌ `mountain rocks` |
| 坡、阜、丘 | 土坡小丘 | `earthen slope`, `small hill` | 需判断空间 |
| 远山、群山 | 自然远山 | `distant mountains`, `natural hills` | — |
| 峰、岳、峦 | 大山 | `mountain peak`, `majestic mountain` | — |

**判断原则**：
- 园林/庭院内的"山"→ 多为假山
- 原文有"出园"、"登高远望"→ 可有真山远景
- 原文有"墙"、"园"、"院"等边界词→ 封闭空间

### 假山场景强制约束词

**问题**：AI 易将"假山"生成为"自然山野"，必须强制庭院环境。

| 必须包含 | 说明 |
|----------|------|
| `courtyard setting` | 庭院场景 |
| `garden wall visible` | 围墙可见 |
| `brick/stone floor` | 铺地（非自然地面） |
| `enclosed space` | 封闭空间 |
| `man-made arrangement` | 人工排列 |

**假山场景 Prompt 模板**：

```
【假山场景模板】
Chinese courtyard garden interior,
garden wall visible forming enclosure,
artificial rockery stones stacked in courtyard corner,
brick or stone paved floor not natural ground,
enclosed private garden,
man-made landscape design,
courtyard scale intimate not vast,
potted plants visible（盆栽可见）,
traditional Chinese garden setting,
```

### 封闭空间背景模板

```
【背景约束】（封闭空间通用）
within enclosed walls,
no distant mountains, no distant landscape,
boundary visible (wall/fence/building),
enclosed setting,
```

### 园艺风格对照

| 风格 | 特征 | Prompt 关键词 |

| 元素 | 说明 | 负面关键词 |

| 建筑类型 | 规模特征 | Prompt 关键词 |
|----------|----------|--------------|
| 民居 | 单层，三开间 | `humble cottage`, `simple dwelling` |

### 常见误解修正

| 误解 | 原因 | 修正 |
|------|------|------|
| 贵族园林 → 皇宫 | 过度渲染 | `private garden`, `not imperial scale` |
| 正殿/大殿 → 皇宫大殿 | 词义误解 | `main hall within private garden` |
| 园林 → 皇家御苑 | 园林无规模感 | `private noble garden`, `family estate` |

### 规模判定原则

| 原文线索 | 判定 | Prompt |
|----------|------|--------|
| "省亲别墅"/"贵妃省亲" | 贵族园林（为迎接贵妃而建，但仍是私宅） | `private noble garden`, `not imperial` |
| "府"/"宅"/"园" | 私人宅邸 | `private residence` |
| 原文无"皇家"/"御"/"圣" | 默认私人规模 | `private scale`, `not royal` |

### 园林内"正殿/大殿"的正确理解

| 原文用词 | 正确理解 | Prompt 关键词 |
|----------|----------|--------------|
| 正殿 | 园林内主体建筑 | `main hall within garden`, `reception pavilion` |
| 大殿 | 可能是园林主体 | `garden main hall`, `not throne hall` |
| 行宫 | 皇家临时住所 | 需判断是否皇家语境 |

## 使用方法

### 查询流程

1. 阅读原文，识别古文意象
2. 在本表查找对应意象
3. 使用"视觉关键词"列的英文术语
4. 检查"常见误解"列避免错误

### 示例

**原文**："只见千百竿翠竹遮映"

**查询**：翠竹千百竿 → `thousands of emerald bamboo, bamboo grove as main focus`

**Prompt 片段**：
```
thousands of emerald bamboo screening buildings,
bamboo grove as main visual focus,
```

**避免**：❌ 几株竹子、普通植物

## 【新增】植物造型约束

### 传统园林 vs 西方园艺

| 对比项 | 中国传统园林 | 西方园艺 |
|--------|-------------|----------|
| 植物形态 | 自然生长，不修剪 | 修剪成几何形状 |
| 绿篱 | 天然编织（竹、树枝） | 人工修剪整齐 |
| 排列 | 疏密有致 | 整齐对称 |
| 美学 | 天人合一 | 人工控制 |

### 禁止的西方园艺元素

| 元素 | 说明 | 负面关键词 |
|------|------|-----------|
| 修剪灌木 | 方形/球形造型 | `no trimmed geometric hedges` |
| 绿篱球 | 圆形灌木 | `no topiary balls` |
| 整齐绿篱 | 直线排列 | `no manicured hedges`, `no formal garden` |
| 修剪草坪 | 现代草坪 | `no manicured lawn` |
| 几何花坛 | 对称图案 | `no geometric flower beds` |

### 传统植物描述

| 原文用词 | 正确理解 | Prompt 关键词 |
|----------|----------|--------------|
| 分畦列亩 | 农田划分，非修剪灌木 | `divided vegetable plots`, `farm fields arranged` |
| 青篱 | 天然编成的篱笆 | `woven natural fence`, `branches woven into fence` |
| 绿柳周垂 | 垂柳自然下垂 | `weeping willow branches drooping naturally` |
| 翠竹 | 自然丛生 | `bamboo growing in natural clumps` |
| 杏花如霞 | 自然开花 | `apricot trees flowering naturally` |

### 菜畦 vs 灌木区分

| 要素 | 菜畦 | 修剪灌木 |
|------|------|----------|
| 形态 | 土地划分 | 植物造型 |
| 边界 | 垄沟分隔 | 修剪边缘 |
| 内容 | 蔬菜作物 | 观赏植物 |
| 关键词 | `vegetable plots`, `farm fields` | ❌ 应禁止 |

### 植物约束模板

```
【植物造型约束】
natural untrimmed plants,
no geometric hedges, no topiary,
no manicured garden design, no formal garden,
traditional Chinese garden style,

```

## 扩展规则

### 遇到表中未列意象

1. **原文优先**：直接翻译原文描述
2. **模糊处理**：不确定时用通用词
3. **记录反馈**：添加新意象到本表

### 通用原则

| 原则 | 说明 |
|------|------|
| 不添加 | 原文未提及的元素不出现 |
| 不省略 | 原文的修辞、排比、数量保持 |
| 不过度推演 | 避免添加原文隐含但不明确的内容 |
| 数量精确 | 使用原文的数量词 |
